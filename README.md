# 🔨 Text Forge
**An Interactive Typography Animation Engine**

Text Forge is a concept project developed to streamline the workflow between design and frontend implementation. It provides a visual interface for "forging" complex CSS typography animations that are lightweight, accessible, and performant.

## 🎓 Academic Goals
* **Performance Optimization:** Reducing reliance on heavy motion assets (GIFs/MP4s) by utilizing pure CSS3.
* **Accessibility:** Ensuring animated text remains selectable by cursors and readable by screen readers.
* **Workflow Efficiency:** Creating a "no-code" bridge for designers to generate production-ready CSS keyframes.

## ✨ Core Features
* **Foundry:** Curated library of 20 Google Fonts + Custom Font URL injection.
* **The Forge:** Real-time control over weight, size, spacing, and line height.
* **Motion Library:** 20 distinct CSS animations with adjustable speed and letter-staggering.
* **Interactive Logic:** Toggle between continuous loops or user-triggered hover states via radio controls.
