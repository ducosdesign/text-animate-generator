# 🔨 Text Forge

**Text Forge** is a high-performance web typography tool designed for designers and developers who want to bridge the gap between static text and immersive micro-interactions. Create, style, and animate typography with granular control, then forge it into production-ready code.

---

## 🚀 The Vision
Most web text is static. Adding animation usually requires heavy GIF exports or complex JS libraries. **Text Forge** solves this by generating lightweight, pure CSS/HTML snippets that maintain accessibility and SEO while delivering high-impact visual motion.

## ✨ Core Features

* **Custom Foundry:** Access a curated library of 20+ Google Fonts or inject your own custom `.ttf/.otf` via URL.
* **Precision Styling:** Adjust font weight, letter spacing, and line height with real-time preview.
* **Motion Library:** 20 built-in CSS animations including *3D Flips*, *Rainbow Fades*, and *Kinetic Jumps*.
* **Intelligent Logic:** * **Loop Mode:** For constant brand presence.
    * **Hover Trigger:** For interactive micro-moments.
* **Instant Export:** Download as a high-res PNG or copy the self-contained "Forge Code" to paste into any website.

## 🛠 Tech Stack
* **Core:** HTML5, CSS3 (Custom Variables & Keyframes), Vanilla JavaScript.
* **Engine:** `html2canvas` for static exports.
* **Foundry:** Google Fonts API integration.

## 📖 Case Study: "PixelPath Studios"
**Problem:** A digital agency saw a 15% drop in Lighthouse scores due to heavy animated assets.
**Forge Solution:** Replaced GIFs with Text Forge CSS snippets.
**Result:** 22% improvement in load speeds and 14% increase in user engagement on hero sections.
